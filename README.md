【模块职责】 
负责小车底盘运动、电机驱动、黑线循迹、速度闭环与路径执行。

【硬件配置】
电机驱动：TB6612
循迹传感器：8路红外循迹阵列
控制芯片：TI MSPM0G3507
测速：电机编码器（速度闭环）

【核心功能】
电机驱动与速度闭环控制
黑线循迹与偏差计算
循迹PID控制（直线、弯道、路口稳定）
原地旋转找线逻辑
丢线保护与自动回线
路口/节点识别
终点精准停车
底盘锁死/解锁安全机制

【使用引脚】 
• 电机模块使用引脚：PA8, PA26, PB9, PB19
• 循迹模块使用引脚：PA1, PA2, PA3, PA4, PA6, PA7, PA10, PA14

## 引脚分配说明
### 1. 电机驱动模块（输出）
用于控制左右轮电机的正反转，适配 TB6612 驱动芯片。

| 定义名 | 实际引脚 | 功能说明 | 典型接线目标 |
|--------|----------|----------|--------------|
| AIN1 | GPIOA_8 | 左轮方向控制信号1 | 驱动板 AIN1 |
| AIN2 | GPIOA_26 | 左轮方向控制信号2 | 驱动板 AIN2 |
| BIN1 | GPIOB_9 | 右轮方向控制信号1 | 驱动板 BIN1 |
| BIN2 | GPIOB_19 | 右轮方向控制信号2 | 驱动板 BIN2 |

**工作逻辑**
- 正转：AIN1高，AIN2低
- 反转：AIN1低，AIN2高
- 刹车：AIN1低，AIN2低

---

### 2. 8路循迹传感器（输入）

| 定义名 | 实际引脚 | 位置说明 | 功能说明 |
|--------|----------|----------|----------|
| IR1 | GPIOA_4 | 最左侧传感器 | 检测左边缘 |
| IR2 | GPIOA_6 | 左中侧传感器 | 辅助左偏判断 |
| IR3 | GPIOA_7 | 左内侧传感器 | 近距离左偏检测 |
| IR4 | GPIOA_14 | 中左传感器 | 中心偏左循迹 |
| IR5 | GPIOA_1 | 中右传感器 | 中心偏右循迹 |
| IR6 | GPIOA_2 | 右内侧传感器 | 近距离右偏检测 |
| IR7 | GPIOA_3 | 右中侧传感器 | 辅助右偏判断 |
| IR8 | GPIOA_10 | 最右侧传感器 | 检测右边缘 |

**工作逻辑**
- 黑线：输出低电平 0
- 白区：输出高电平 1

---


【代码模块职责说明】
• motor.c：负责电机的初始化、启停、正反转与速度控制，通过驱动芯片实现小车前进/后退/转向。
• track.c：负责读取8路循迹传感器数据，计算赛道偏差，为 PID 控制提供误差信号。
• main.c：负责系统初始化、主循环调度，调用循迹模块的误差值，通过 PID 算法控制电机转向，实现自动循迹。
